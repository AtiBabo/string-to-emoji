# string to emoj
string to emoji는 python으로 작성되었습니다.

# 사용방법
파이썬 파일을 실행시킨 뒤 "입력 : "부분에 지원되는 글들을 작성해주세요.

----------
##### 지원되는 글들 : 알파벳(대문자, 소문자), 숫자, +, -, *, /, =, !, ?

그 후 출력이 된 글을 복사 한 후 Discord에 붙여넣기 해주세요.
